Your LogiQ Voice Lab backend is basically a text → speech pipeline.

Not magic.
Not a full AI model.

It’s a workflow system that:

1. receives text


2. sends it to Microsoft Edge neural voices


3. receives audio


4. streams MP3 back to your app



Here’s the real backend architecture in simple terms.


---

🧠 HOW THE BACKEND ACTUALLY WORKS

Frontend (your app UI)
        ↓
API Route (/api/tts)
        ↓
Voice Processing Logic
        ↓
Microsoft Edge Neural Voice Service
        ↓
MP3 Audio Stream
        ↓
Frontend Audio Player


---

🔥 STEP-BY-STEP BREAKDOWN

1. User clicks “Generate Voice”

From frontend:

textarea content

selected voice

speed settings


are sent to backend.

Example request:

{
  "text": "Mansa Musa crossed the Sahara...",
  "voice": "en-US-GuyNeural",
  "rate": "0%"
}


---

2. Backend API route receives request

Example route:

/api/tts

This route:

validates input

prevents empty requests

checks allowed voice list

sanitizes text


Usually done with:

Zod

TypeScript validation



---

3. Backend converts text into SSML

This is VERY important.

Microsoft voices don’t just read plain text.

They use:

SSML

(Speech Synthesis Markup Language)

Example:

<speak>
  <voice name="en-US-GuyNeural">
    <prosody rate="0%">
      Mansa Musa crossed the Sahara.
    </prosody>
  </voice>
</speak>

This controls:

voice

pacing

pauses

emotion

pronunciation



---

🔥 THIS is why your voice sounds cinematic

Not because of “AI magic.”

Because:

neural voice model

SSML formatting

pacing structure



---

4. Backend opens WebSocket connection

This is the most technical part.

Your backend:

connects to Microsoft Edge TTS service

sends SSML

receives audio chunks


The service streams:

binary audio data

back continuously.


---

5. Backend rebuilds MP3 stream

Your server:

combines audio chunks

converts them into proper audio response

streams them back to frontend


Response type:

audio/mpeg


---

6. Frontend receives audio

Browser:

creates temporary audio URL

loads player

auto-plays narration


Example:

const audioUrl = URL.createObjectURL(blob)


---

7. Download button saves MP3

Frontend creates:

logiq-voice-2026.mp3

Then downloads it locally.
